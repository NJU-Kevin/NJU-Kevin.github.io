背景

Glauber动力学 & Kawasaki动力学

考虑物理里研究铁磁性和温度关系的Ising模型，在$\mathbb{Z}^d$上的点，每个点都有一个自旋，要么向上要么向下，微观状态的构型则是将这所有点的自旋状态组合在一起。具体而言，考虑状态空间$\mathcal { X } = \{ \pm 1 \} ^ { \mathbb { Z } ^ { d } }$，赋予离散概率测度：

其中$\eta = \left( \eta _ { x } \right) \in \mathcal { X } , \ \beta = \frac { 1 } { k T } > 0, \ d \eta _ { x } = \big ( \delta _ { 1 } + \delta _ { - 1 } \big ) \big ( d \eta _ { x } \big )$

进而$\eta ( t ) = \{ \eta _ { x } ( t ) \} _ { x \in \mathbb { Z } ^ { d } }$便是状态空间$\mathcal{X}$上的Markov过程。Ising模型也可以这样看：视作随机游走，$\eta_x(t)=+1$ iff $x$处存在一个粒子；$\eta_x(t)=-1$ iff $x$处没有粒子。

Kawasaki动力学：粒子的总数不变。

Glauber动力学：粒子的数量可以变化，可以出现新的粒子也可以消失。

上面所述的Ising模型是一种微观模型，但当粒子数趋于无穷大时，随机密度场将会演化为一个确定的流体力学方程（非线性PDE）。通过拉长空间（微观下粒子是离散的，宏观下更关注粒子的连续坐标位置）和时间（微观下的粒子跳动速度极快，时间尺度也要拉长进行观察），每个非常小的宏观区域里，粒子都达到了局部平衡状态，而这个局部平衡状态只和某些特定的参数有关，而这些参数正好由非线性PDE或者SDE表现出来。
