概要

这篇文章将对《Real and Complex Analysis》（亦即所谓的“Papa Rudin”）的课后习题进行思路的阐述，并重点讨论每道题所需用到的知识点以及定理，而不会详细地写出每道题的严格数学证明（这件事早在2019年就有教授做过了，而且证明写得十分详细）。每道题我都将讲述我个人的解题思路以及想到的相关知识，梳理的同时并对这本名著的基本内容进行概述。

Chapter 1 Abstract Integration

Exercises 1

Does there exist an infinite a-algebra which has only countably many members ?

这道题问是否存在拥有可数个元素的无穷$\sigma$代数，看上去很宽泛，貌似很难具体入手。事实上，$\sigma$代数是一个很大的集族，它的性质也很好，包含了空集和全集，同时只要一个集合在里面，那么其补集也在；还对可数并封闭。需要注意的是，在书中关于$\sigma$代数定义的Comments部分，揭示了其关于可数交也封闭，且任何两元素的差集也在$\sigma$代数中的性质，这往往是容易忽视的一点，但的确很好用。

因此既然这个$\sigma$代数没有确切的模样，不如从其中一个元素入手。可以找一个非空的属于该$\sigma$代数的集合$E$，那么$E$和$E^c$中一定有一个是无限集，不妨为$E$，$E$本身也形成了一个子可测空间（只需要让$E$中的$\sigma$代数是原来的$\sigma$代数中的集合与$E$的交集即可），然后$E$中又可以取真子集，这样递归地下去，得到一个无穷递降子集列，每个集合都是无限集，且都在$\sigma$代数中，由于$\sigma$代数对差集也封闭，这个链集中的相邻集合的差集也在$\sigma$代数中，这些差集与自然数构成了一一对应，$\sigma$代数还需对这些差集的任意并封闭，这就与自然数的子集构成了一一对应，而自然数的子集个数是不可数的，从而$\sigma$代数也不可数。

与自然数建立双射的过程并不困难，加上$\sigma$代数需要满足的可数并性质便可以让其中元素不可数，因此这道题告诉我们无穷$\sigma$代数是很“大”的.

Exercise 3

Prove that if f is a real function on a measurable space X such that { x : f(x) > r} is measurable for
every rational r, then f is measurable.

看上去是很常规的概念题，使用集合论应该就可以解决。做的时候发现的确是这样，利用定义，证明$\mathbb{R}$中开集的原像可测，由题意可以知道左闭右开区间的原像可测，而开区间可以用左开右闭区间进行可数逼近，于是$f$可测.
